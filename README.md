# d-comments

## Requirements

- node.js ^16.0.0 (<https://nodejs.org/ja/>)
- yarn (<https://yarnpkg.com/>)
- Google Chrome (Latest) or Microsoft Edge (Latest)

### Command Line

#### Installation

```bash
yarn install
```

#### Build

```bash
yarn build
```

#### Publish

```bash
yarn production
```

#### Lint

```bash
yarn lint
```
